# Music Library Manager

A **web application for managing songs, albums, and artists** built with **ASP.NET Core Razor Pages** and **Entity Framework Core**.

This project allows users to:

- Add, edit, delete, and view songs
- Manage albums and artists
- Search and filter songs by artist, album, or title
- View detailed information for each song

---

## 🛠️ Technologies Used

- **.NET 7 / .NET Core** – Razor Pages for server-side rendering  
- **Entity Framework Core** – Database ORM  
- **SQLite** – Local database (`music.db`)  
- **Bootstrap 5** – Responsive UI  
- **C#** – Backend logic  
- **Git & GitHub** – Version control  

---

## 📂 Features

### Songs
- Create, Read, Update, Delete (CRUD) operations  
- Search songs by title or filter by artist/album  
- Validation for required fields and release year  

### Albums & Artists
- Manage albums and artists  
- Add, edit, and delete functionality  
- Relationship between songs, albums, and artists  

### User Interface
- Responsive design using Bootstrap 5  
- Error handling with friendly messages  
- Dropdowns for selecting albums and artists
